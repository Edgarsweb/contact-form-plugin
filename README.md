# contact-form-plugin
Simple contact form plugin for testing
After uploading, installing and activating the plugin, you will see on the left admin panel Plugin's name "Contact form by Edgar".
You can click on it and copy the shortcode [edxweb_form] for activating this plugin in your wordpress site.
You will receive mail notifications to your admin email, so please make sure it has been set to the email address, you want to get notifications on.
